# Alphabera Global Website

This repository contains the Alphabera Global marketing website built with React and Vite.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `VITE_WHATSAPP_NUMBER` in `.env` if you want the contact modal to open WhatsApp with your preferred number
3. Run the app:
   `npm run dev`
